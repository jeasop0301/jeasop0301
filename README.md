## About

Apple 플랫폼(**macOS · iOS**)과 저지연 시스템을 만드는 인디 개발자입니다.
**Swift**로 앱을, **Rust**로 성능이 중요한 부분을 씁니다.

- **정직한 설계** — 측정 불가는 `0`이 아니라 '측정 불가'로.
- **로컬 우선 · 프라이버시** — 계정도, 원격 텔레메트리도 없이.
- **작고 뾰족한 도구** — 하나를 제대로 하는 메뉴바 앱과 CLI.

> **Now** — Kelvin의 배터리 코치와 betterparsec의 저지연 전송을 다듬는 중.
