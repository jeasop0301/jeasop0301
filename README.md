### 👋 소개 · About

Apple 플랫폼(macOS · iOS)과 저지연 시스템/네트워킹을 만드는 인디 개발자입니다.
**Swift**로 맥/아이폰 앱을, **Rust**로 성능이 중요한 시스템을 씁니다.

- 🌡️ **정직한 소프트웨어** — 측정 불가는 `0`이 아니라 '측정 불가'로. 조작된 숫자는 없습니다.
- 🔒 **로컬 우선 · 프라이버시** — 계정 없음, 원격 텔레메트리 없음.
- 🧩 **작고 뾰족한 도구** — 하나를 제대로 하는 메뉴바 앱과 CLI를 좋아합니다.

<sub>Building things for Apple platforms and low-latency systems. Swift for apps, Rust for the hard parts. Local-first and honest by design.</sub>
